# Education Logos

This directory contains the logos for educational institutions.

## Required Images

1. **CAS Logo** - `cas_logo.png`
   - Institution: Institute of Automation, Chinese Academy of Sciences
   - Recommended size: 60x60 pixels or larger (square aspect ratio)

2. **USTB Logo** - `ustb_logo.png`
   - Institution: University of Science and Technology Beijing
   - Recommended size: 60x60 pixels or larger (square aspect ratio)

## Notes

- If the logo files are not found, a placeholder will be displayed
- PNG format with transparent background is recommended
- Keep file sizes small for faster loading
